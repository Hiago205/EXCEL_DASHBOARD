# Dashboard de Vendas

Projeto desenvolvido para análise de vendas de uma empresa do setor de papéis para embalagens.

## Dados
Base fictícia, porém realista, contendo:
- Data
- Produto
- Região
- Vendedor
- Quantidade
- Preço
- Faturamento

## Ferramentas
- Excel
- Power BI

## Objetivo
Transformar dados brutos em informações visuais para apoio à tomada de decisão.
